## Принципы разработки

1. **Поэтапная разработка** — каждый блок завершается и тестируется отдельно
2. **Модульность** — компоненты и эффекты изолированы по файлам
3. **Прозрачная отладка** — ключевые действия логируются в консоль
4. **Fallback-подход** — визуальные эффекты имеют безопасные значения по умолчанию
5. **Версионирование** — устаревшие материалы переносятся в `archive/`
6. **Чистота и архив** — то, что не используется в продакшене (тяжёлые анимации, старые концепты), складываем в `archive/` вместо удаления.
