## 📐 Размеры и отступы

### **Grid System (8px)**
- **Base unit**: 8px
- **Spacing**: 8px, 16px, 24px, 32px, 48px, 64px
- **Breakpoints**: 320px, 768px, 1024px, 1440px

### **Типографика**
- **H1**: 48px (desktop), 36px (mobile)
- **H2**: 36px (desktop), 28px (mobile)
- **H3**: 24px (desktop), 20px (mobile)
- **Body**: 16px (desktop), 14px (mobile)
- **Caption**: 12px

### **Компоненты**
- **Button height**: 48px
- **Input height**: 48px
- **Card padding**: 24px
- **Section padding**: 80px (desktop), 40px (mobile)

## 🎯 Интерактивные элементы

### **Hover состояния**
- **Кнопки**: Подсветка + подъем
- **Карточки**: Тень + масштаб
- **Ссылки**: Подчеркивание
- **Навигация**: Активное состояние

### **Анимации**
- **Появление**: Fade in + slide up
- **Переходы**: 0.3s ease-in-out
- **Loading**: Spinner + skeleton
- **Scroll**: Parallax эффекты

## 📱 Адаптивные особенности

### **Mobile (< 768px)**
- Стекинг в одну колонку
- Увеличенные touch targets
- Упрощенная навигация
- Оптимизированные изображения

### **Tablet (768px - 1024px)**
- 2 колонки для портфолио
- Средние размеры элементов
- Гибридная навигация

### **Desktop (> 1024px)**
- 3+ колонки для портфолио
- Полная навигация
- Расширенные анимации
- Hover эффекты
