# Блог

Структура блога использует гибридный подход: метаданные в JSON, контент в отдельных HTML файлах.

## Структура

- `articles.json` - метаданные всех статей
- `posts/` - HTML файлы статей
- `assets/` - изображения для статей

## Добавление новой статьи

1. Создать HTML файл в `posts/` (например, `2024-12-20-my-article.html`)
2. Добавить запись в `articles.json` со следующими полями:
   - `id` - уникальный идентификатор (slug)
   - `title` - заголовок статьи
   - `date` - дата публикации (YYYY-MM-DD)
   - `tags` - массив тегов
   - `excerpt` - краткое описание (150-200 символов)
   - `contentFile` - путь к HTML файлу статьи
   - `status` - "published" или "draft"
   - `readTime` - примерное время чтения в минутах
   - `icon` - иконка из `assets/icons/`

## Использование утилит

- `scripts/add-article.js` - интерактивное добавление статьи
- `scripts/validate-blog.js` - валидация структуры блога
