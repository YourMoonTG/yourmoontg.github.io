## Текущая структура

```
bio-website-v2/
├── Корневые файлы
│   ├── index.html
│   ├── blog.html
│   ├── README.md
│   └── start-site.cmd
│
├── Основные папки
│   ├── css/          # Стили
│   ├── js/           # JavaScript модули
│   ├── assets/       # Ресурсы (иконки)
│   ├── projects/     # Страницы проектов
│   ├── blog/         # Блог (статьи, метаданные)
│   ├── api/          # API сервер
│   ├── scripts/      # Утилиты Node.js
│   └── docs/         # Документация
│
├── Служебные папки
│   ├── archive/      # Архив старых материалов
│   └── backups/      # Бэкапы (игнорируется Git)
│
└── Конфигурация
    └── .gitignore    # Правила игнорирования
```

## Принципы организации

1. **Модульность** - каждый компонент в отдельном файле
2. **Логическая группировка** - связанные файлы рядом
3. **Разделение по типам** - CSS, JS, HTML в отдельных папках
4. **Документация рядом с кодом** - README в каждой папке где нужно
5. **Архив вместо удаления** - старые материалы в archive/
